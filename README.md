# 燕园易骑 (PKU Cycle-Recycle Hub)

> 将"僵尸车"转化为新生资产——北京大学校园自行车循环回收平台

## 项目简介

**燕园易骑** 是一个校园自行车循环回收平台，旨在解决北京大学校园内废旧自行车堆积的问题。平台连接即将毕业离校的学生（需要处置自行车）和新生或其他师生（需要低廉的代步工具），通过数字化流程实现车辆的高效回收与再利用。

### 核心数据
- 🚲 **76%** 的新生希望使用回收自行车
- ♻️ **89%** 的毕业生愿意捐赠或出售自行车
- 🌱 减少资源浪费，降低购车成本，打造绿色循环校园生态

### 目标用户
- **供应方（卖家/捐赠者）**：即将毕业离校的学生
- **需求方（买家）**：刚入学的新生及校内其他有代步需求的师生
- **管理员（Admin）**：负责线下车辆存放、成色鉴定及核销的学生志愿者或校园管理人员
- **主负责人（SuperAdmin）**：项目的核心管理者，拥有最高权限

## 功能特性

### 🏠 门户与宣传
- 清晰的"捐赠/出售"与"寻找座驾"入口
- 实时展示回收数据（累计回收数量、减少金属浪费等）
- 公告栏：管理员发布平台公告，所有用户可见

### 🚲 车辆回收模块（供应方流程）
1. **车辆登记**：上传照片，填写品牌、新旧程度、意向价格（可"免费捐赠"）
2. **管理员审核**：批准或驳回，驳回时给出理由
3. **时间段选择**：管理员提出可选时间段，卖家选择并确认
4. **线下交易**：双方按约定时间进行线下交车
5. **交易完成**：管理员确认，卖家可进行评价

### 🔍 智能匹配与预约（需求方流程）
1. **搜索与筛选**：按价位、成色范围筛选自行车
2. **实时库存**：车辆状态实时更新，避免冲突
3. **预约锁定**：选择心仪车辆，创建预约申请
4. **时间段确认**：管理员提出时间段，买家选择并确认
5. **线下提车**：按约定时间提车
6. **交易完成**：管理员确认，买家可进行评价

### ⏰ 时间段管理系统
- 管理员为预约创建多个可选时间段
- 用户从可用时间段中选择（支持日历时间选择器）
- 支持"drop-off"（卖家交车）和"pick-up"（买家提车）两种类型
- 交易倒计时：首页显示距离交易的剩余时间

### 💬 社区广场（论坛）
- 发布帖子：分享建议、使用心得或问题
- 互动功能：评论、点赞、收藏
- 删除功能：发帖人可删除自己的帖子和评论

### 📩 私信系统
- 管理员与用户私信沟通
- 用户之间互相发送私信
- 支持文字和图片消息
- 未读消息提醒（红点提示）

### 👤 个人中心
- 查看发布的自行车、预约记录
- 查看收藏的帖子
- 查看私信记录
- 时间段选择：选择管理员提供的时间段
- 历史记录：查看已完成的交易

### 🔧 管理后台
- **仪表盘**：显示待处理任务、倒计时、统计数据
- **车辆收购**：
  - 审核预约：审核卖家登记的车辆
  - 收购确认：确认车辆交易完成
  - 库存管理：查看和管理库存车辆
- **车辆交付**：
  - 提车预约：为买家预约提出时间段
  - 交付确认：确认买家提车完成
  - 交付管理：管理历史预约记录
- **用户管理**：查看所有用户信息
- **车辆管理**：直接操作自行车状态（设为库存、已售等）或删除车辆
- **公告管理**：发布、编辑、删除平台公告

## 使用指南

### 1. 注册与登录

**用户注册**：
1. 访问网站首页，点击"注册"
2. 填写邮箱、密码、姓名
3. 注册成功后即可登录

### 2. 卖家流程（出售/捐赠自行车）

1. **登录**后，点击首页"捐赠/出售"按钮
2. **填写车辆信息**：
   - 上传车辆照片
   - 填写品牌、新旧程度（1-10 成）
   - 填写意向价格（可选择"免费捐赠"）
3. **等待审核**：管理员会在后台审核您的车辆
4. **选择时间段**：
   - 审核通过后，您会收到私信通知
   - 进入"个人中心" → "时间段选择"
   - 从管理员提供的时间段中选择一个
5. **线下交车**：按约定时间到指定地点交车
6. **交易完成**：管理员确认后，您可以对服务进行评价

**取消交易**：在登记到线下交易的任何阶段，您都可以取消交易。

### 3. 买家流程（寻找座驾）

1. **登录**后，点击首页"寻找座驾"按钮
2. **浏览与筛选**：
   - 查看库存中的自行车
   - 按价格、成色筛选
3. **锁定车辆**：
   - 选择心仪的车辆，点击"预约"
   - 车辆状态变为"锁定中"
4. **等待时间段**：
   - 管理员会提出可选时间段
   - 您会收到私信通知（首页个人中心图标显示红点）
5. **选择时间段**：
   - 进入"个人中心" → "时间段选择"
   - 从管理员提供的时间段中选择一个
6. **线下提车**：按约定时间到指定地点提车
7. **交易完成**：管理员确认后，您可以对服务进行评价

**取消预约**：在交易完成前，您可以随时取消预约。

### 4. 社区广场使用

1. 点击首页"社区广场"进入论坛
2. **发帖**：点击"发布帖子"，填写标题和内容（可添加图片）
3. **互动**：
   - 点赞：点击帖子下方的点赞按钮
   - 评论：在帖子底部输入评论内容
   - 收藏：点击收藏按钮，可在个人中心查看收藏的帖子
4. **删除**：发帖人可删除自己的帖子和评论

### 5. 私信使用

1. 点击首页"私信"进入私信页面
2. **查看消息**：
   - 左侧显示对话列表
   - 右侧显示具体消息内容
   - 未读消息显示红点提示
3. **发送消息**：
   - 选择对话对象
   - 输入文字或上传图片
   - 点击发送
4. **系统通知**：管理员发送的系统通知也会显示在私信中

### 6. 管理后台使用（仅管理员）

1. 登录后，点击首页"管理后台"进入
2. **仪表盘**：
   - 查看待处理任务数量（红点提示）
   - 查看交易倒计时
   - 点击"跳转"按钮快速进入相应功能
3. **车辆收购**：
   - 审核预约：审核卖家登记的车辆，批准或驳回
   - 收购确认：确认车辆交易完成
   - 库存管理：查看和管理库存车辆
4. **车辆交付**：
   - 提车预约：为买家预约提出时间段
   - 交付确认：确认买家提车完成
   - 交付管理：管理历史预约记录
5. **用户管理**：查看所有用户信息
6. **车辆管理**：直接操作自行车状态或删除车辆
7. **公告管理**：发布、编辑、删除平台公告

## 快速开始

### 环境要求
- Node.js 18+
- Python 3.9+
- npm 或 yarn

### 本地开发

```bash
# 克隆项目
git clone https://github.com/LilyHanZhang/PKU-Campus-Cycle-Cycle.git
cd PKU-Campus-Cycle-Cycle

# 安装前端依赖
cd frontend
npm install
cp .env.example .env.local  # 配置环境变量
npm run dev

# 安装后端依赖（新终端）
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### 访问地址
- 前端：http://localhost:3000
- 后端 API：http://localhost:8000
- API 文档：http://localhost:8000/docs
