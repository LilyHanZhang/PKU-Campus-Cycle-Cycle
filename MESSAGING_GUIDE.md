# 私信功能使用指南

## 问题：看不到联系人和"新建对话"按钮

### 原因分析
从截图看，页面显示"暂无会话"，但**右上角应该有"新建对话"按钮**。

### 解决方案

#### 方法 1: 强制刷新浏览器缓存
**Vercel 可能还在部署中，或者浏览器缓存了旧版本**

1. **Windows/Linux**: 按 `Ctrl + Shift + R` 或 `Ctrl + F5`
2. **Mac**: 按 `Cmd + Shift + R`
3. 或者打开开发者工具（F12），右键刷新按钮，选择"清空缓存并硬性重新加载"

#### 方法 2: 检查 Vercel 部署状态
1. 访问 https://vercel.com
2. 查看你的项目部署状态
3. 如果显示 "Building" 或 "Ready"，等待部署完成
4. 部署完成后，等待 1-2 分钟让 CDN 更新

#### 方法 3: 清除浏览器缓存
1. 打开浏览器设置
2. 清除浏览数据（缓存和 Cookie）
3. 重新访问网站

## 正确使用流程

### 步骤 1: 登录账号
访问：https://pku-campus-cycle-cycle.vercel.app
- 输入邮箱和密码登录
- 或注册新账号

### 步骤 2: 进入私信页面
- 从个人中心点击"私信"按钮
- 或直接访问：https://pku-campus-cycle-cycle.vercel.app/messages

### 步骤 3: 点击"新建对话"按钮
**位置：页面右上角，"返回个人中心"按钮左侧**

按钮样式：
- 紫色背景
- 白色文字 "新建对话"
- 左侧有 "+" 图标

### 步骤 4: 选择用户
1. 点击"新建对话"后，弹出模态框
2. 从下拉列表中选择要联系的用户
3. 点击"开始对话"按钮

### 步骤 5: 发送消息
1. 在聊天窗口中输入消息
2. 点击发送按钮或按 Enter 键
3. 对方会在他们的会话列表中看到新消息

## 功能说明

### 新建对话
- 显示所有注册用户列表
- 自动过滤掉自己
- 选择用户后自动发送第一条消息（"你好！"）
- 创建新的会话

### 给自己发消息（笔记功能）
- 可以选择给自己发送消息
- 用作个人笔记、备忘录、待办事项
- 在会话列表中会显示自己的头像

### 搜索联系人
- 在搜索框中输入用户名
- 快速找到特定联系人
- 支持模糊搜索

## 常见问题

### Q1: 为什么看不到"新建对话"按钮？
**A**: 
- 可能是浏览器缓存了旧版本
- Vercel 部署还未完成
- 尝试强制刷新（Cmd+Shift+R）

### Q2: 为什么用户列表是空的？
**A**:
- 系统中可能没有其他用户
- 需要先注册多个账号才能看到其他用户
- 检查网络连接是否正常

### Q3: 如何测试私信功能？
**A**:
1. 注册两个不同的账号
2. 用账号 A 登录
3. 点击"新建对话"，选择账号 B
4. 发送消息
5. 用账号 B 登录查看

## 技术检查

### 检查 API 是否正常
访问：https://pku-campus-cycle-cycle.onrender.com/users/
- 需要登录后访问
- 应该返回用户列表 JSON 数据

### 检查前端版本
打开浏览器开发者工具（F12）
- Console 标签查看是否有错误
- Network 标签查看 API 请求是否成功
